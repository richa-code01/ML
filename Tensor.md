# Tensor Basics

# What is a Tensor?

A tensor is an N-dimensional array used to store numerical data.

---

# Tensor Dimensions

## Scalar
0D Tensor

### Example

```text
5
```

---

## Vector
1D Tensor

### Example

```text
[1, 2, 3, 4]
```

---

## Matrix
2D Tensor

### Example

```text
[[1, 2],
 [3, 4]]
```

---

# Axis / Rank

- Number of dimensions in a tensor
- Also called Rank

---

# Example

```text
[1, 2, 3, 4]
```

Rank = 1
