# 1. Introduction to Machine Learning

## What is Machine Learning?
Machine Learning (ML) is a way of teaching computers to learn patterns from data so they can perform tasks on similar unseen data without being explicitly programmed for every step.

Instead of manually writing rules for every situation, we allow machines to discover patterns automatically.
