
### 2. KNN (K-Nearest Neighbors)

A supervised learning algorithm used for:
- Classification
- Regression

Predicts output using nearest data points.

---

### Working of KNN

#### Step 1: Choose K

Example:

```text
K = 5
```

Means:
- find 5 nearest neighbors

> K is a hyperparameter.

---

#### Step 2: Calculate Distance

Usually uses **Euclidean Distance**.

$$
d=\sqrt{(x_2-x_1)^2+(y_2-y_1)^2}
$$

Can work in:
- 2D
- 3D
- n-dimensional space

---

#### Step 3: Sort Distances

Arrange neighbors from:
- nearest → farthest

---

#### Step 4: Majority Voting

Example:

```text
Yes → 3
No  → 2
```

Prediction:

```text
Yes
```

Example:
```text
Person can become hacker
```

---

### KNN for Regression

Instead of majority voting:
- take mean/average of nearest neighbors

Example:

```text
Nearest salaries:
20k, 25k, 30k, 35k, 40k

Prediction = Mean
```

---

### Choosing K Value

- Small K → noisy, overfitting
- Large K → smoother, underfitting

#### Important

K is usually kept:
- odd number

Reason:
- avoids tie during majority voting

Example:

```text
K = 4
Yes = 2
No  = 2
```

Tie occurs.

---

### How to Find Best K?

Using **Cross Validation**.

#### Cross Validation

1. Try multiple K values

```text
K = 1,3,5,7,9...
```

2. Train + evaluate model
3. Compare evaluation metrics:
   - Accuracy
   - Precision
   - Recall
   - F1 Score

4. Select K giving best performance.

---

### Feature Scaling in KNN

KNN is distance-based.

Features with larger values dominate distance.

#### Example

```text
Age = 20
Salary = 200000
```

Salary dominates calculations.

So feature scaling is required.

---

### Common Scaling Methods

- Standardization
- Min-Max Scaling

---

### Advantages

- Simple and intuitive
- No training phase(Instance based learning)
- Works for multi-class problems

---

### Disadvantages

- Slow for large datasets
- Sensitive to noise
- Requires feature scaling
