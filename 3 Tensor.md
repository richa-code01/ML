# Tensor Basics

# What is a Tensor?

A tensor is an N-dimensional array used to store numerical data.

---

# Tensor Dimensions

## Scalar
0D Tensor

### Example

```text
5
```

---

## Vector
1D Tensor

### Example

```text
[1, 2, 3, 4]
```

---

## Matrix
2D Tensor

### Example

```text
[[1, 2],
 [3, 4]]
```

---

# Axis / Rank

- Number of dimensions in a tensor is called Axis
- Also called Rank

---

# Example

```text
[1, 2, 3, 4]
```

Rank/Axis = 1

---
# No. of dimensions of a vector = no. of elements in it.

#Example

```text
[1, 2, 3, 4]
```
Dimension of this vector = 4 (data represented in 4 coordinates)
